### ECE 532 HW4: Histogram Thresholding
## Requirements
This program was built on Ubuntu 20.04 using GNU Makefile.

## How to Build and Run
To build and install the program simply invoke

```bash
$ make install
```

to run the program gui execute
```
$ hdcr --gui
```
