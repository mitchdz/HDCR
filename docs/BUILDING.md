# Building

Instructions for building and configuring hdcr

## Step 1 - Dependencies
###opencv2
on Ubuntu 20.04, install using:
```bash
sudo apt install libopencv-dev python3-opencv
```

## Step 2 - Building

Run the `make` command.

```sh
$ make
```

## (OPTIONAL) Step 3 - Run

To execute the program with the demo picture, execute:
```sh
$ make run
```
